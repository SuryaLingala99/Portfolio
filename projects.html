AmethystTrader: High-Frequency Predictive Engine

The AmethystTrader addresses the challenge of identifying exploitable anomalies in high-frequency equity markets. At its foundation, the system implements a real-time ingestion layer utilizing Alpaca WebSockets to process asynchronous JSON streams of multi-timeframe bar data. This ingestion mechanism is designed to handle high message throughput while maintaining a persistent cache for low-latency feature calculation.

A primary technical accomplishment of this project is the integration of an automated hyperparameter optimization engine. To account for shifting market volatility regimes, the system executes a RandomizedSearchCV sequence across a rolling 5-day historical window. This process leverages Scikit-learn to discover the optimal configuration for an XGBoost ensemble model. To prevent look-ahead bias—a common failure point in financial modeling—the system utilizes TimeSeriesSplit for cross-validation, ensuring the model is strictly trained on chronologically preceding iterations.

The feature engineering pipeline transforms raw price and volume metrics into a high-dimensional vector space. Key engineered features include dynamic Volume-to-SMA ratios, RSI divergence, and Bollinger Band width compression. These vectors are then passed through the gradient-boosted decision tree ensemble to output a directional probability map. The engine is hosted on a containerized FastAPI backend on Render, with a secure environment variable management system for managing sensitive API credentials.

RealEstateSwarm: Multi-Agent Extraction Architecture

The RealEstateSwarm was architected to solve the complexity of aggregating fragmented property data from highly dynamic web sources. Traditional regex-based or static DOM scraping is ineffective against modern anti-bot detections. This system overcomes these limitations by utilizing LangGraph to orchestrate a distributed swarm of specialized AI agents, each tasked with a specific phase of the data synthesis pipeline.

The infrastructure utilizes a cluster of headless Playwright browsers to execute complex navigational sequences, bypassing protective layers through adaptive header rotation and simulated human interaction flows. Once a target page is successfully ingested, an Analysis Agent powered by large language models executes a multi-step extraction protocol. This agent is trained to identify and structure property attributes from unstructured text, such as school zone ratings, square footage context, and taxation history.

To ensure the integrity of the resulting dataset, the swarm implements a recursive validation loop. Extracted data points are cross-referenced against SERP indexed snapshots to confirm accuracy before being stored in a structured relational format. The entire system is built for deployment within a Dockerized environment, enabling seamless scaling of agent nodes across cloud providers like Render and Vercel. This architecture represents a shift from brittle scraping to a resilient, self-correcting agentic data ingestion pipeline.
